# prek

## Sources

- https://github.com/j178/prek
- https://prek.j178.dev/

## Installation

Install using `mise`:
```shell
mise use prek
```

Get prek version:
```shell
prek --version
```

## Usage

Install the prek git hook:
```shell
prek install
```

Run hooks:
```shell
prek run
```

List hooks:
```shell
prek list
```

Uninstall the prek git hook:
```shell
prek uninstall
```

Auto-update pre-commit config to the latest repos' versions:
```shell
prek auto-update
```
