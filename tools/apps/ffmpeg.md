# ffmpeg

A complete, cross-platform solution to record, convert and stream audio and video.

## Sources

- https://ffmpeg.org/
- https://github.com/FFmpeg/FFmpeg

## Usage

Extract subtitle from a mkv file:
```shell
ffmpeg -i video.mkv -map 0:s:0 subs.srt
```
