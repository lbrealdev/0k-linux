# Just

## Sources

- Website: https://just.systems/
- GitHub Repository: https://github.com/casey/just

## Usage

Installing `just`:
```shell
curl --proto '=https' --tlsv1.2 -sSf https://just.systems/install.sh | sudo bash -s -- --to /usr/local/bin
```

Get `just` version:
```shell
just --version
```

List recipes:
```shell
just --list
```

Run `just` fmt:
```shell
just --unstable --fmt --check
```
