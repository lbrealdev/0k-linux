# sdkman

### Sources

- Website: https://sdkman.io
- GitHub Repository: https://github.com/sdkman/sdkman-cli

## Usage

Installing `sdkman`:
```shell
curl -sSL "https://get.sdkman.io" | bash
```

```shell
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

Install java:
```shell
sdk list java

sdk install java
```

Install maven:
```shell
sdk list maven

sdk install maven
```
