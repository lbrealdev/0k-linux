# apt

apt - command-line interface

## Usage

```shell
apt search <package>
```

```shell
apt show <package>
```

```shell
apt-cache search <package>
```

```shell
apt-cache show <package>
```

```shell
apt-cache policy <package>
```

```shell
apt-cache depends <package>
```

```shell
apt list --installed
```

```shell
apt update
```

```shell
apt upgrade
```
