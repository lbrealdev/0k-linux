# date

```shell
date +%s
```

```shell
date +%Y-%m-%d -d 'today'
```

```shell
date +%Y-%m-%d -d '2 day ago'
```

```shell
date '+%Y-%m-%d %H:%M:%S %Z'
```

Date rfc 3339 `date`, `seconds` and `ns`:
```shell
date --rfc-3339=seconds
```

```shell
date -u '+%Y-%m-%d %H:%M:%S'
```

```shell
date '+%Y-%m-%d %H:%M:%S'
```
